# Multi-Agent Startup Simulator

Multi-Agent Startup Simulator is a personal project about building a startup from the ground up through a team of collaborating AI-style agents. The idea is to simulate how different roles in a real company think, argue, and coordinate while turning one prompt into a practical business outcome.

The core scenario is simple:

> Launch a fitness app

From that one idea, the agents debate and shape:

- Business plans
- Marketing plans
- Pricing strategy
- Product roadmaps

## Project Vision

This project is designed around a startup-style multi-agent system where each agent has a clear responsibility:

- CEO Agent
- Marketing Agent
- Finance Agent
- Product Agent
- Sales Agent

Each agent contributes a different perspective so the final output feels like a real internal company discussion instead of a single generic answer.

## Learning Approach

This repository is primarily an experiment in project-based learning. The goal is to build the actual logic and important decisions by hand, with AI used only for learning support or boilerplate code when needed. The core logic, structure, and important implementation details are intended to be created manually.

## Current State

Right now, the project contains a basic Python/Tkinter starting point. The next step is to expand it into a real multi-agent simulator with separate roles, message exchange, and startup-planning output.

## Example Output Flow

1. A user submits a startup idea.
2. The CEO Agent coordinates the discussion.
3. The Product Agent defines features and roadmap.
4. The Marketing Agent proposes positioning and launch ideas.
5. The Finance Agent suggests pricing and cost assumptions.
6. The Sales Agent adds go-to-market and customer acquisition ideas.
7. The system returns a combined startup strategy.
